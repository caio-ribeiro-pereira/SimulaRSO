Projeto SimulaGP
====================================================================

É um aplicativo que simula o comportamento dos principais algoritmos
de escalonamento de processos concorrentes (FC-FS, Round Robin, SJF, SRT) 
implementados em qualquer sistema operacional.
Totalmente modelado em Java e sua interface gráfica foi escrita em JavaFX
utilizando a IDE Netbeans 6.9.

O Objetivo principal desse projeto é disponibilizar uma ferramenta de apoio web,
para estudos sobre a disciplina de Sistema Operacionais, de forma que o projeto
seja facilmente integrado a qualquer Sistema de ensino a distância (EAD).

Este projeto é o nosso trabalho de conclusão de curso que será contribuído
para a Universidade Católica de Santos - www.unisantos.br.
